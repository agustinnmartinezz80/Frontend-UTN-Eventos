# Frontend UTN - Eventos

Aplicación frontend desarrollada con React + Vite para gestionar eventos.  
Permite registrar usuarios, iniciar sesión, verificar cuenta por email y administrar eventos (crear, editar, eliminar y visualizar).


##  Tecnologías utilizadas
- React
- React Router DOM
- Axios
- Tailwind CSS
- Vite
- Context API para autenticación 


## Estructura del proyecto
src/
 ├── components/
 ├── pages/
 ├── context/
 ├── hooks/
 ├── services/
 └── App.jsx


 ## Funcionalidades
- Autenticación

- Registro

- Inicio de sesión

- Verificación por email

- Logout

- Sesión persistente con Context API

## Gestión de eventos

- Crear evento

- Editar evento

- Eliminar evento

- Ver listado de eventos

- Ver detalle de evento

## Rutas protegidas

- Implementadas mediante ProtectedRoute
